Где можно получить эту специальность?

Обратная связь

Специалист способен выполнять следующую профессиональную работу:

  • техник-программист на предприятиях;
  • специалист финансово-экономического отдела;
  • администратор;
  • аналитик
  • ИТ-менеджер;
  • разработчик компьютерных программ.

Специалист по экономической кибернетике может занимать следующие должности:

  • разработчик пакетов прикладных программ (ППП) общего и проблемного назначения;
  • специалист по внедрению новейших информационных технологий;
  • администратор базы данных (АБД);
  • администратор управленческих информационных систем (УИС);
  • инженера-программиста;
  • инженера по автоматизированных компьютерных системах;
  • разработчика информационных и экспертных систем в экономике, финансово-кредитной сфере, менеджменте и маркетинге;
  • инженера по программному обеспечению компьютеров;
  • экономиста (по планированию, по финансовой работе, по бухгалтерскому учету и анализу хозяйственной деятельности);
  • руководителя аналитического центра по обработке экономической, финансовой и учетной информации;
  • руководителя подразделения компьютерных услуг;
  • руководителя малого предприятия.
